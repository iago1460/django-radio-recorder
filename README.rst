================
RadioCo recorder
================
**A recorder program**.

This program communicates with RadioCo

Main Program
============

https://github.com/iago1460/django-radio

Website
-------

http://radioco.org/


Installation
============

Requirements
------------

- Python 2.7

Install requirements::

	pip install -r requirements.txt

Install avconv::

    sudo apt-get install libav-tools


Usage
-----

run command::
	
	python main.py
